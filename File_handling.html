File Handling:
A file is a location on the hard disk which can be used for storing data. A file should be used for storing the data when we have a small amount of data.
The files are classified into two types
1)Text Files
2)Binary Files
In a file we can perform either reading or writing operations. To perform the reading or writing operations on the file, the file must be opened. To open a file we take the help of open().
Syntax: open(filename, mode)
The various modes are:
r: This mode can be used for reading the data from the specified file. If the specified file is not available then we get FileNotFoundError. r is the default mode.
w: This mode can be used for writing the content into the specified file. If the specified file does not exist then it will create a new file and then write the content into that file but if the specified file already exists then it will overwrite the content.
a: This mode can be used for writing the content into the specified file. If the specified file does not exist then it will create a new file and then write the content into that file but if the specified file already exists then it will append the content.
x: This mode can be used for writing the content into the specified file. If the specified file does not exist then it will create a new file and then write the content into that file but if the specified file already exists then we get FileExistsError.
r+: This mode can be used for reading and writing the data into the specified file.
w+: This mode can be used for writing and reading the data from the specified file.
a+: This mode can be used for reading and appending the data into the specified file.
The above modes are applicable to text files. If we want to apply the modes to the binary files, then we need to suffix b to all the above modes(rb, wb, ab, xb, r+b, w+b, a+b).
Once we perform the reading and writing operations on the file, we need to close the file.To close the file we take the help of close().


Writing Operations: To write the data into a file, we have the following 2 methods.
write(): This method can be used to a write a string into a file.
writelines(): This method can be used to multiple lines of string into a file.


Program: to write the content into a file with 'w' mode
f = open('xyz.txt','w')
f.write('hello friends')
f.write('good morning')
f.write('this is file handling concept in python')
f.write('thank you')
f.close()
print('data is stored into the file')


To perform reading and writing operations, we need to open the file. Once we perform reading and writing operations, we have to close the file explicitly by using close().
If we want to close the file automatically after the operations are performed then we need to take the help of with keyword.
Program: to write the content into a file with 'w' mode
with open('xyz.txt','w') as f:
   f.write('hello friends')
   f.write('good morning')
   f.write('this is file handling concept in python')
   f.write('thank you')
print('data is stored into the file')
Once the control comes out of the with suite, the file will be closed automatically.

Reading Operations: To read the data from a file, we have the following 4 methods.

read(): This method will read the total data from the file.
readline(): This method will read only one line from the file.
readlines(): This method will read all the lines from the file into a list.
read(n): This method will read n characters from the file.


Program: to read the entire file
f = open("abcd.txt")
text = f.read()
print(text)
f.close()


Program: to read the entire file

   text = f.read()
   print(text)


Program: to read one line from a file

When we are writing data into a file and if we do not close the file, then the content will not be placed into the file. If we want to place the content into the file without closing, then we need to take the help of flush().
flush(): This method will clear the buffer and place the content into the file.
Program: to clean the buffer
f = open('xyz.txt','w')
f.write('hello friends')
f.write('good morning')
f.write('this is file handling concept in python')
f.write('thank you')
f.flush()
print('data is stored into the file')

zipfile module
ZIP is an archive file format that supports lossless data compression. A ZIP file is a single file containing one or more compressed files, providing a best solution to make large files smaller and keep related files together.
Program: to compress a list of files
from zipfile import *
f = ZipFile('files.zip','w',ZIP_DEFLATED)
f.write('abcd.txt')
f.write('xyz.txt')
f.write('file.txt')
f.close();


Program: to uncompress a file
from zipfile import *
f = ZipFile('files.zip','r',ZIP_STORED)
names = f.namelist()
for filename in names:
   print(filename)
   f1 = open(filename,'r')
   print(f1.read())
   print()


Program: to display the details of the files available in zip file
from zipfile import ZipFile
import datetime
with ZipFile('files.zip', 'r') as zp:
   for info in zp.infolist():
     print(info.filename)
     print('Modified : ' + str(datetime.datetime(*info.date_time)))
     print('System : ' + str(info.create_system) + '(0 = Windows, 3 = Unix)')
     print('ZIP version : ' + str(info.create_version))
     print('Compressed : ' + str(info.compress_size) + ' bytes')
     print('Uncompressed : ' + str(info.file_size) + ' bytes\n')


Pickling and Unpickling
Pickling: It is a process converting an object into a stream of bytes.
Unpickling: It is a process converting a stream of bytes into object.
dump(): This function is used to serialize an object.
load(): This function is used to de-serialize an object.
The dump() and load() are available in pickle module.

Program: to perform pickling and unpickling
import pickle
class Customer:
   def __init__(self, cid, cname, caddr):
     self.cid = cid
     self.cname = cname
     self.caddr = caddr

   def display(self):
     print("cid :",self.cid)
     print("cname :",self.cname)
     print("caddr :",self.caddr)

with open("data.txt",'wb') as f:
   c = Customer(123,'Manoj','hyderabad')
   pickle.dump(c,f)

with open("data.txt",'rb') as f:
     obj = pickle.load(f)
     obj.display()
