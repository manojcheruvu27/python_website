Iterators
Iterator is an object that can be iterated upon. An iterator object which will return one element at a time.


Program: to display the values available in the list
data = ['abcd','mnop','qwer','xyz']
for x in data:
   print(x)

1)Every sequence that we have in python by default contains an iter attribute to iterate through the elements available in the sequence. Every iterator contains a next attribute to get the next element available in the sequence.
2)If we want to develop our own iterator, we need to develop a class containing the implementation of __iter__() and __next__().
3)The __iter__() method must return an object of an iterator itself.
4)The __next__() method must return the next element in the sequence. After reaching the end, it must raise StopIteration exception.
Program: to develop our own iterator to display the values available in the list
class Remote:
   def __init__(self):
      self.channels = ['maa','gemini','star','discovery']
      self.index = -1
   def __iter__(self):
      return self
   def __next__(self):
      self.index += 1
         if self.index == len(self.channels):
         raise StopIteration
      return self.channels[self.index]
r = Remote()
itr = iter(r)
print(next(itr))
print(next(itr))
print(next(itr))
print(next(itr))

Generators
1) Generator is a function with yield statement instead of a return statement.
2) If a function contains at least one yield statement, it becomes a generator function.
3) Both yield and return will return some value from a function.
4) The difference is that a return statement terminates the function where as a yield statement pauses the function saving all its data and later on continues from there on the next call.
5) Generators are a simple way of creating iterators.
6) A generator is a function that returns an iterator which we can iterate and return one value at a time.
7) Generators will improve the performance of the application.


Program: to generate multiple string values using generator
def languages():
   yield 'java'
   yield 'python'
   yield '.net'
   yield 'scala'
   yield 'sas'
l = languages()
print(next(l))
print(next(l))
print(next(l))
print(next(l))
print(next(l))

Program: to generate Fibonacci series using generator
def fibonacci():
   n1 = 0
   n2 = 1
   count = 0
   while count<=10:
      yield n1
      n1, n2 = n2, n1+n2
      count = count + 1
values = fibonacci()
for x in values:
   print(x)
Deep Copying and Shallow Copying
Using the = operator we can copy a list into another list. When we are copying a list into another list by using =, it will not create another list instead both will refer to the same list.

Program: to implement copying of a list into another list
first_list = [1, 2, 3, 4, 5, 6]
second_list = first_list


print('First List :', first_list)
print('Second List :', second_list)
print('ID of First List :', id(first_list))
print('ID of Second List :', id(second_list))
first_list[2] = 8
print('First List :', first_list)



print('Second List :', second_list)
print('ID of First List :', id(first_list))
print('ID of Second List :', id(second_list))
second_list[2] = 9
print('First List :', first_list)



print('Second List :', second_list)
print('ID of First List :', id(first_list))
print('ID of Second List :', id(second_list))

Sometimes we want to have the values in the original list unchanged and modify only the values in the new list or vice versa in such case there are two ways to create duplicates copies.
1)Shallow Copy
2)Deep Copy
To perform shallow copy or deep copy we need to use copy module. To show shallow and deep copy we need to have nested lists.

Shallow Copy: To perform shallow copying we need to use copy() available in copy module.
In shallow copy it does not create a duplicate copy of nested objects, instead it just copies the reference of nested objects.
Program: to implement shallow copy

import copy
first_list = [1, 2, [3, 4], 5]
second_list = copy.copy(first_list)



print('First List :', first_list)
print('Second List :', second_list)
print('ID of First List :', id(first_list))
print('ID of Second List :', id(second_list))



first_list[2][0] = 6
first_list.append(7)
print('First List :', first_list)
print('Second List :', second_list)
print('ID of First List :', id(first_list))
print('ID of Second List :', id(second_list))



first_list[1] = 8
print('First List :', first_list)
print('Second List :', second_list)
print('ID of First List :', id(first_list))
print('ID of Second List :', id(second_list))
