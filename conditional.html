Conditional statements:
1)Usually the statements in python are executed sequentially.
2)To disturb the flow of execution or to change the sequence we use conditional statements.
3)There are 3 types of conditional statements.
i)if.
ii)if else.
iii)else if ladder.
4)Conditional statements in python use suite with 4 spaces indentation.
