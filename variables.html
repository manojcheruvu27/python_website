Variable Names

1) A variable can have a short name (like x and y) or a more descriptive/clear name (age, carname, total_volume).

2)Unlike any other language Declaring datatype is not required while declaring a variables in python.The syntax goes as follows.
SYNTAX: Variable_name = value;
Python is smart enough to guess the datatype of the variable.

Rules For declaring Python variables:

i)A variable name must start with a letter or the underscore character
ii)A variable name cannot start with a number
iii)A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
iv)Variable names are case-sensitive (age, Age and AGE are three different variables)
